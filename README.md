# Frontend part of dimplom work

Based on react-redux-webpack-typescript boilerplate

## Installation

```
$ npm ci
```

## Running

```
$ npm start
```

## Build

```
$ npm run build
```

## Deploy (to the [GitHub Pages](https://pages.github.com/))

```
$ npm run deploy
```

## Format code (using [Prettier](https://github.com/prettier/prettier))

```
$ npm run prettier
```
